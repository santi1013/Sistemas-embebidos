Detailed Description

Interfaz pública del driver de acceso a la microSD.

Este archivo declara las funciones necesarias para inicializar la tarjeta microSD, verificar su estado de montaje y realizar operaciones básicas sobre archivos, como abrirlos, escribir líneas de texto y cerrarlos.

El objetivo de este módulo es encapsular el acceso al sistema de archivos para que el resto del proyecto pueda trabajar con funciones simples y no dependa directamente de la configuración SPI ni del montaje del volumen.

esp_err_t sdcard_init ( void )

Inicializa la microSD y monta el sistema de archivos.

Esta función configura el bus SPI, intenta detectar la tarjeta y monta el sistema de archivos FAT en el punto de montaje definido por el driver.

Debe ejecutarse antes de cualquier operación de apertura o escritura de archivos.

Returns
  • ESP_OK si la tarjeta fue inicializada y montada correctamente.
  • Código de error si falla la inicialización del bus SPI o el montaje.

Inicializa la microSD y monta el sistema de archivos.

Esta función configura el host SPI, inicializa el bus físico usando los pines definidos en el proyecto y luego intenta montar la tarjeta SD en la ruta /sdcard.

La lógica interna sigue esta secuencia:

  • mostrar información de depuración de los pines usados,
  • configurar el montaje del sistema FAT,
  • preparar el host SPI y el bus físico,
  • inicializar el bus SPI,
  • asociar la tarjeta al pin CS,
  • montar la microSD en el sistema de archivos virtual del ESP32.

Si todo sale bien, se habilita la bandera interna g_sd_mounted y el sistema queda listo para abrir archivos.

Returns
  • ESP_OK si la tarjeta fue montada correctamente.
  • Código de error si falla la inicialización del SPI o el montaje de la SD.

Abre un archivo en la microSD en modo append.

Esta función verifica primero que la tarjeta esté montada. Si la SD está disponible, intenta abrir el archivo indicado en modo "a", lo que permite agregar contenido al final sin borrar los datos existentes.

Su lógica es preparar el acceso al archivo de salida que será usado por el sistema de registro.
