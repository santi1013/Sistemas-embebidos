Detailed Description

Estructura que representa un registro listo para guardar en archivo.

Esta estructura se utiliza después del procesamiento de los datos del sensor. Contiene la marca de tiempo asociada al registro y una línea de texto ya formateada para ser escrita directamente en el archivo de log dentro de la microSD.

Su lógica es desacoplar el procesamiento numérico de la etapa de escritura, permitiendo que una tarea genere el contenido y otra lo almacene.
